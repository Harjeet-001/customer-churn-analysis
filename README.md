# Customer Churn Analysis

## Problem Statement
Customer churn directly impacts revenue and long-term business sustainability.  
This project analyzes customer behavior to identify key drivers of churn and provides actionable insights to improve customer retention.

## Dataset
- Source: Telco Customer Churn Dataset (Kaggle)
- Records: 7,032 customers
- Target variable: Churn (Yes / No)

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- Excel (data inspection)
- GitHub

## Key Analysis Steps
1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Churn segmentation by contract, tenure, charges, and demographics
4. Dashboard creation for executive-level insights

## Key Insights
- Month-to-month contract customers churn significantly more than long-term contracts
- Customers in the first 0–12 months have the highest churn risk
- Senior citizens churn at a much higher rate than non-seniors
- Higher monthly charges are associated with increased churn
- Long-tenure customers show strong retention behavior

## Business Recommendations
- Incentivize customers to move from month-to-month to long-term contracts
- Implement early-tenure retention programs within the first year
- Offer targeted plans or support for senior customers
- Review pricing strategy for high monthly charge segments
